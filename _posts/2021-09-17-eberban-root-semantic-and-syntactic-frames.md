```
Eberban Root Semantic Frames

Legend:
 - Xd: distributive concrete (entity or event)
 - Xc: collective concrete (entity or event)
 - P0: proposition
 - P1: property
 - P2: relation
 - SX: sequence of entities
 - S0: sequence of events
 - S1: sequence of properties
 - S2: sequence of relations
 - Nn: number

MUI:   nullary

MI:    Xd
MERE:  Xd Xd
VAN:   Xd Xd Xd
ZAOR:  Xd Xd P0
ZDIR:  Xd Xd P1
TCON:  Xd Xd P2
ZDE:   Xd Xd Nn
BLAE:  Xd Xc
MARA:  Xd P0
FUIR:  Xd P0 Xd
MORE:  Xd P1
ZMI:   Xd P1 Xc
SFO:   Xd P2
FNIU:  Xd Nn

ZA:    Xc
TSEN:  Xc Xd
MARNE: Xc Xc
CONE:  Xc Xc S2
CONI:  Xc Xc Nn S2
GZA:   Xc P0
MOLAI: Xc P1
TUN:   Xc P1 Xc
SPEU:  Xc P2
BEIN:  Xc Sx
CANE:  Xc S1
CANI:  Xc Nn S1
MENE:  Xc Nn Xc Sx

SENI:  P0 Xc
BZEN:  P0 P0
SEN:   P0 P0 P0

TSE:   Nn Xc
CNI:   Nn Sx P1
CENI:  Nn S0
MUAL:  Nn Nn

CNE:   Sx P1
CENE:  S0
```

```
Eberban Root Syntactic Frames

Legend:
 - X: concrete (entity or event)
 - P: predicate (proposition, property or relation)
 - S: sequence
 - N: number


MUI:  nullary

MI:   X           subtypes: ZA
MERE: X X         subtypes: BLAE, TSEN, MARNE
VAN:  X X X
ZAOR: X X P       subtypes: ZDIR, TCON
ZDE:  X X N
CONI: X X N S
MARA: X P         subtypes: MORE, SFO, GZA, MOLAI, SPEU
FUIR: X P X       subtypes: ZMI, TUN
BEIN: X S         subtypes: CANE
FNIU: X N
MENE: X N X S
CANI: X N S

SENI: P X
BZEN: P P
SEN:  P P P

TSE:  N X
CENI: N S
CNI:  N S P
MUAL: N N

CENE: S
CNE:  S P
```
